The Hitchhiker's Guide to LUA.

"Getting Started" 

                    
                        --As in most coding languages, the most basic function possible is printing "Hello World".
                        --In LUA, this is done with the following code:
                        
                        print("Hello World")

"Variables" 

                    
                        --Once you know how to print something, you can move into assigning variables.
                        --A variable can be identified by the following code:
                        
                        local variable = 100

"Using/Changing Variables" 

                    
                        --Once a variable is set, it can be utilized or changed with the following:
                        
                        local variable = 100

                        --Printing a variable
                        print(variable + 100) -- prints 200

                        --Changing a variable
                        variable = variable + 100
                        print(variable) -- prints 200

"Tables" 

                    
                        --A table can be set in a way similar to a variable, but can contain a various amoutn of them.
                        --The code below shows how to identify a table and whatever is inside of it:
                        
                        local table = {100, 200, 300}

"Using/Changing Tables" 

                    
                        --Utilizing a table is similar to utilizing a variable.  Below shows how to do so:
                        
                        local table = {100, 200, 300}

                        --Printing a part of a table
                        print(table[1]) -- prints 100

                        --Changing a part of a table
                        table[1] = 200
                        print(table[1]) -- prints 200
