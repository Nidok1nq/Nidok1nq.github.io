The Hitchhiker's Guide to LUA.

"Getting Started"

As in most coding languages, the most basic program you can write is the "Hello World" program. This is a simple program that prints the words "Hello World" to the screen. In LUA, this is done with the following code: 

                    
                        print("Hello World")

"Variables"

Once you have down the basic "print" you can move into printing vaiables. A variable is identified as shown in the code below: 

                    
                        local variable = 100

"Using/Changing Variables"

Once a variable is set, it can be utilized or changed, as shown below: 

                    
                        local variable = 100

                        --Printing a variable
                        print(variable + 100) -- prints 200

                        --Changing a variable
                        variable = variable + 100
                        print(variable) -- prints 200
